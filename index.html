Nigerian Banking API

Live

Production-deployed banking system covering the full backend lifecycle: ACID-safe transactions, event streaming, observability, and CI/CD — not a tutorial project, a system built to run in production and survive concurrent load.

  • Row-level locking (FOR UPDATE) prevents race conditions on concurrent fund transfers
  • IDOR-proofed: every account query scoped by user_id at the SQL layer
  • Kafka event streaming (KRaft mode) publishes transaction events after every committed write
  • Redis caching with graceful degradation — a cache outage never breaks a committed transaction
  • Prometheus + Grafana observability stack, scraped live on Railway
  • Multi-stage Docker build, GitHub Actions CI running Jest/Supertest on every push
  • React/TypeScript frontend consuming the live API end to end
Node.js TypeScript Express PostgreSQL Redis Kafka Docker Prometheus Grafana GitHub Actions Railway React
